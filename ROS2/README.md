# ROS2 Onboarding Guide

ROS2(Robot Operating System 2) 학습을 위한 예제 코드와 상세 설명 가이드입니다.

- **Distribution:** ROS2 Kilted Kaiju
- **OS:** Ubuntu 24.04 LTS (Recommanded)

---

## 📂 저장소 구조

이 저장소는 효율적인 학습을 위해 두 개의 영역으로 나뉘어 있습니다.

* **[guides/](./guides):** ROS2의 기본 개념, Jazzy 버전의 특징, 설치 방법 등을 정리한 문서 폴더입니다.
* **[examples/](./examples):** 실제 구동 가능한 예제 소스 코드가 담긴 폴더입니다.

## 🚀 시작하기

1. **환경 설정:** `docs/` 폴더 내의 설치 가이드를 먼저 확인해 주세요.
2. **코드 실행:** `examples/` 폴더의 각 하위 프로젝트에 포함된 README를 참고하여 빌드 및 실행을 진행합니다.
